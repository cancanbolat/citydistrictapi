Turkish City - District API
--------

Features
--------
- Laravel
-  MySQL

## App URLs

- http://127.0.0.1:8000
    - All cities: `/api/cities`
    - Get city by id: `/api/city/33`
    - All districts: `/api/districts`
    - Get district by id: `/api/district/718`
    - City districts: `/api/citydistricts/33`
